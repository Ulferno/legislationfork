Chain of Command

The Chain of Command is a very important aspect of the Security Department. Each position is defined within this chain. The chain of command must be followed at all times. When contacting a command member regarding something, follow this chain of command:

3 Officers > 3 Inspectors > Chief

Do not contact the Security Chiefs for matters easily solvable by Security Officers or Inspectors.

Security Chief SC-4 (Max 2)#

Description#

The Security Chief is seen as the Director of the Security Department. They are responsible for maintaining the departmentâ€™s overall core. The Security Chiefs are also able to promote Security Officers to Security Inspector. The Security Chiefs are chosen by the Overseer of the Security Department. There may only ever be two Security Chiefs at one given time.

Requirements#

In order to be promoted to Security Chief, you must meet the following requirements:

  • Handpicked by the Security Overseer

Security Inspector SC-3 (Max 6)#

Security Inspectors are seen as the core management of the Security Department, primarily overseeing operations carried out by Security Officers, as well as drafting up necessary information to present to the Security Chiefs. Security Inspectors are chosen by the Security Chiefs and are promoted from Security Officers. There may only be seven Security Inspectors at one given time unless overruled by the Overseer of the Security Department.

In order to be promoted to Security Inspector, you must meet the following requirements:

  • 3 weeks as a Security Officer
  • Actively contributing to the Security Departmentâ€™s command team as an Officer
  • Have hosted a minimum of 15 trainings
    • 5 Combat Trainings
    • 5 Standard Trainings
    • 5 Cadet Orientations
  • Noticed and chosen by the Security Chiefs

Security Officer SC-2 (Max 25)#

Security Officers are considered the Medium Command of the Security Department. They are responsible for leading on-duty security personnel while on site, and they may be seen as a point of guidance. Security Officers are also responsible for hosting trainings and doing administrative work such as disciplining personnel and handling promotions. There may only be twenty-five Security Officers at any given time.

In order to be promoted to Security Officer, you must meet the following requirements:

  • Actively contributing in-game.
  • Helpful within the SD discord, maintaining sufficient activity.
  • 2 Standard trainings hosted as a Security Sentry.
  • 2 Combat Trainings/Scenario Trainings hosted as a Security Sentry.
  • At least 2 weeks as a Security Sentry.
  • Must have the Host role within the SD discord server.
  • Handpicked by the Security Inspectors with approval from the Security Chiefs.

Security Sentry SC-2 (Max 30)#

Security Sentry is the highest non-command position, and are in-training to become Security Officers. Security Sentry are able to host trainings with supervision, as well as supervise Security Cadets during tests, and authorise them inside the Class D containment zone. They have slightly more authority then a regular guard whilst on-site.

In order to be promoted to Security Sentry, you must meet the following requirements:

  • 3 weeks as a Security Guard.
  • 45 hours in-game in total.
  • 1 Standard Training co-hosted.
  • 1 Combat Training/Scenario Training co-hosted.
  • 2 Uproars attended as Guard.
  • 2 backup request/rallies attended as Guard.
  • 1 Red Heron attended as Guard.
  • Utmost Competency, and loyalty to the Security Department.

Once you have completed the first 7 of these, fill out a promotion request in the correct channel and wait patiently. Security Inspector+ attends to promotion requests for Guard > Sentry.

Fair reminder: This rank cannot be easily obtained, reaching the first 7 requirements will not grant you this rank. Only those most competent and dedicated will attain the position.

Security Guard SC-2#

Security Guards are the 2nd low-ranking position. Security Guards are eligible to co-host trainings hosted by Security Officers. And are still able to apply for the Military Police and Combat Medic Subdivisions as well as contend in SRU Tryouts. Security Guardâ€™s have the same power on-site as Security Sentry, meaning theyâ€™re able to supervise Security Cadets during tests, and authorise them inside the Class D containment zone.

In order to be promoted to Security Guard, you must meet the following requirements:

  • 2 weeks as a Security Junior.
  • 30 hours in-game in total.
  • 1 Standard Trainings passed as Junior.
  • 2 Combat Training/Scenario Training passed as Junior.
  • 3 Uproars attended in total.
  • 2 backup request/rallies attended in total.
  • 1 Red Heron attended.
  • A written evaluation after completing the above requirements (Junior > Guard Exam)

Once you have completed the first 7 of these, fill out a promotion request in the correct channel and wait patiently.

Security Junior SC-2#

Security Juniors are considered the third low-ranking position. This position is eligible to apply for the Military Police and Combat Medics subdivisions.

In order to be promoted to Security Junior, you must meet the following requirements;

  • 3 days (72 hours) within the Security Department
  • 1 Cadet Orientation Passed.
  • 1 Combat Training/Scenario Training Passed.
  • 1 Standard Training Passed.
  • 3 hours in-game in total.
  • 2 Uproars attended.
  • A written evaluation after completing the above requirements (Cadet > Junior Exam)

Once you have completed the first 6 of these, fill out a promotion request in the correct channel and wait patiently.

Security Cadet SC-1#

Security Cadets are considered to be the lowest ranking position in the Security Department. Security Cadets must pass a Cadet Orientation in order to go in-game in the Security Department team.

warning

All Security Cadets, upon joining, have a probationary period of 2 weeks to be promoted to junior. Failure to be promoted by the end of the 2 weeks period will result in EXILE.

(You may request for an extension should you feel you are unable to accomplish your requirements with the given time. Inspector+)

This rank is achieved by passing the Security Department application. The Application Center may be found here.

Promotion Criterion

Members of the Security Department should not be promoted lightly. Members of the Security Department expected to be promoted in accordance with the proper requirements placed on them.

Security Officers are able to recommend personnel for promotions. Security Inspectors are permitted to promote personnel who meet all of the necessary requirements.

Certain requirements may be excused if sufficient proof is shown and deemed necessary by Security Department High Command. Do not resort to this, unless you are really unable to complete said requirement.

Members of the Security Department should only be promoted if they meet the following criterion:

  • They seem plausible to the future of the department.
  • They understand common concepts and have general common sense.
  • They fully understand the Security Department Guidelines & Regulations.
  • They actively contribute via attendance in trainings, communication in Discord, etc.
